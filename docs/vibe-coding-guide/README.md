# Vibe Coding với Claude — Từ Zero đến Sản Phẩm

**Bộ hướng dẫn toàn diện cho người KHÔNG có nền tảng IT**, giúp bạn sử dụng Claude Code + Starter Kit để xây dựng sản phẩm phần mềm chất lượng.

## Bạn sẽ học được gì?

- Hiểu AI/LLM hoạt động thế nào (không cần biết toán hay code)
- Sử dụng Claude Code như một "đội ngũ phát triển" cá nhân
- Áp dụng Starter Kit để đảm bảo chất lượng sản phẩm
- Quy trình Superpowers: từ ý tưởng → sản phẩm hoàn chỉnh

## Đọc ở đâu?

Mở file `.ipynb` bằng **VS Code** (có sẵn Jupyter extension) hoặc xem trực tiếp trên **GitHub**.

## Cách đọc hiệu quả nhất

1. **Mới bắt đầu?** Đọc Part 1 → Part 5 theo thứ tự
2. **Đã biết AI?** Bắt đầu từ notebook 04 (Vibe Coding)
3. **Đã có project?** Nhảy thẳng Part 5 (Thực hành)
4. **Muốn tra cứu?** Dùng mục lục trên để tìm chủ đề cần
